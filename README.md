### Usage:
- View this website on [The Giffy Browser](https://chrisjosh-giffybrowser.netlify.app/) remember to trust the site in your ad blocker if you get net error
- You can also clone this repo and run locally;
    - yarn 
    - yarn start
    
- Run test
    - yarn test
